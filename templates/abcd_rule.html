{{ _('C - Color') }}

{{ _('Color refers to the number of distinct hues visible within the lesion.') }}

{{ _('Distinction of lesion color') }}

{{ _('This symptom is characterized as multi-valued. It describes the presence or absence of allowed ABCD rule colors on the lesion surface. The evaluation involves identifying any number of colors from the set of 6 permitted ones (white, blue, dark brown, light brown, black, and red). An additional criterion is the presence of blue color covering more than half of the lesion. Symptom defining the colours occurring in the lesion therefore takes on a value from 1 to 6. A greater variety in this symptom typically correlates with increased clinical suspicion.') }}

{{ _('D - Diversity of structures') }}

{{ _('Diversity of structures refers to the range of patterns seen within the lesion under dermatoscopic examination.') }}

{{ _('Assessment of structural diversity') }}

{{ _('The last symptom is multi-valued aswell. It describes structures permitted by the ABCD rule on lesion surfaces. Evaluation involves identifying the presence of five main elements: pigment globules, pigment dots, structureless areas, branched streaks, and pigment network. The assessment of the diversity of the mole structures therefore results in assigning a value from 1 to 5 to the symptom of Diversity of structure. Significant diversity of these elements increases the probability that the lesion is a melanoma.') }}

{{ _('Lesion classification based on TDS') }}

{{ _('Classic TDS formula according to Braun-Falco:') }}

TDS = 1.3 * Asymmetry + 0.1 * Border + 0.5 * Color + 0.5 * Diversity of structures

{{ _('Example TDS assessment') }}

{{ _('According to Stolz\'s strategy, the example lesion has no axis of symmetry, one sharp border segment, two colors (dark and light brown), and two structures.') }}

{{ _('TDS calculation example') }}

{{ _('Therefore:') }}

TDS = 1.3 * 2 + 0.1 * 1 + 0.5 * 2 + 0.5 * 2 = 4.7

{{ _('Improved TDS formula') }}

{{ _('As a result of research, scientists developed a more precise TDS formula, assigning individual weights to each color and structure.') }} 

new_TDS = 0.8 * Asymmetry + 0.11 * Border

+ 0.5 * Color_dark_brown + 0.6 * Color_light_brown
